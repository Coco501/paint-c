# Paint-C

This is just a simple version of MS Paint written in C, for fun. I wanted to learn a bit of SDL. Inspired by [Daniel Hirsch](https://www.youtube.com/@HirschDaniel).

### Controls
- `1` - Red
- `2` - Green
- `3` - Blue
- `4` - Black
- `E` - Eraser
- `scroll_wheel` - Adjust brush thickness
